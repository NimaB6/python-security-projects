# Python Security Projects

A collection of small Python projects related to cybersecurity and backend development.

## Projects

### Password Checker
A Python program that checks whether a password is strong or weak.

#### Features
- Check password length
- Check uppercase letters
- Check lowercase letters
- Check numbers
- Check symbols

#### File
`password_checker.py`

#### Example

Input:
```
Hello123
```

Output:
```
Weak password
Problems found:
Password must contain at least one symbol
```
